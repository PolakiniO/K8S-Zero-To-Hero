# Showcase Hub

This folder is the recruiter-facing entry point of the repository.

If you are reviewing this repository for hiring purposes, start here before diving into the full course content.

## What this folder provides

- A fast review path for hiring teams.
- A structured map of deliverables and lab outcomes.
- A reusable template for presenting incident/debugging evidence.
- Public-repo-safe guidance (anonymized identifiers only).

## Scope and privacy

All content in this folder is intentionally anonymized for a public repository:
- no company-identifying names,
- no real account IDs,
- no real IPs or internal endpoints.
