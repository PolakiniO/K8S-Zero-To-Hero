# Resources YAML Mirror

This directory mirrors all YAML manifests in the repository and organizes them by **week** and **lab**.

## Structure

- `week-1/lab-2` → Week 1 Lab 2 manifests
- `week-1/lab-3` → Week 1 Lab 3 manifests
- `week-1/lab-4` → Week 1 Lab 4 manifests
- `week-1/lab-5` → Week 1 Lab 5 manifests
- `week-0/lab-templates` → Shared template manifests from `manifests/templates`

## Source coverage

The following source YAML locations are represented in this mirror:

- `K8S-Lab-Week1/yaml-files/*.yaml`
- `manifests/templates/*.yaml`
